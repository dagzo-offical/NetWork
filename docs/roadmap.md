# Delivery Roadmap

## Phase 1 — Foundation
- Monorepo setup (web/api/shared/content)
- Auth, RBAC, and base schema
- Lesson rendering and locked progression

## Phase 2 — AI Assessment Engine
- Question generation with uniqueness constraints
- Semantic scoring with rubric calibration
- Cooldown and retry orchestration

## Phase 3 — Visual Learning Experience
- Interactive topology, packet flow animations
- TLS handshake visualizer
- Attack/defense scenario labs

## Phase 4 — Enterprise Features
- Analytics dashboard, skill tree, certificates
- QR verification + optional blockchain hash anchoring
- Proctoring and audit trail for final exams
